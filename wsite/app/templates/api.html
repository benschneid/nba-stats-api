API Documentation

Introduction

You can use our free API to access NBA-related data.

  • No email required.
  • No API key required.
  • Contains data from 1946-47 through 2018-2019 seasons.
  • All API parameters must be entered in correct format. If not, 404 Not Found returned.

Game Logs

League Game Log

This endpoint retrieves team box scores from the entire league within a specific season, season mode, and date range.

HTTP Request:

GET https://www.barkleyanalytics.com/api/league_game_log/[season]/[int:playoffs]/[date_from]/[date_to]

Query parameters:

Parameter Format Description
Season 1946-47, ..., 2018-19 The season to collect game logs on.
Playoffs 0 or 1 If 0, collect regular season logs. If 1, collect playoff logs.
Date from/to MM-DD-YYYY Date range within season to collect logs for.

Player Game Log

This endpoint retrieves box score data from a single player within a specific season, season mode, and date range.

HTTP Request:

GET https://www.barkleyanalytics.com/api/player_game_log/[player]/[season]/[int:playoffs]/[date_from]/[date_to]

Query parameters:

Parameter Format Description
Player ID Integer The id of the player to collect game logs on.
Season 1946-47, ..., 2018-19 The season to collect game logs on.
Playoffs 0 or 1 If 0, collect regular season logs. If 1, collect playoff logs.
Date from/to MM-DD-YYYY Date range within season to collect logs for.

Box Scores

Player Box

This endpoint retrieves entire box score of all players for a specific game.

HTTP Request:

GET https://www.barkleyanalytics.com/api/player_box/[game_id]'

Query parameters:

Parameter Format Description
Game ID Integer The game to get data for.

Team Box

This endpoint retrieves box score data from both teams for a specific game.

HTTP Request:

GET https://www.barkleyanalytics.com/api/team_box/[game_id]

Query parameters:

Parameter Format Description
Game ID Integer The game to get data for.
